# AIO_PDCA爆速システム 起動方法

## ⚠️ 重要：ローカルサーバーで起動してください

このシステムは、ブラウザで直接ファイルを開く（`file://`プロトコル）と、セキュリティ制限により動作しません。
**必ずローカルサーバーで起動してください。**

## 起動方法

### 方法1: Python 3を使用（推奨）

1. ターミナルを開く
2. 以下のコマンドを実行：

```bash
cd "月次定例/2026年/1月/analysis_aio/Tier1_AIO/AIO_PDCA爆速システム"
python3 -m http.server 8000
```

3. ブラウザで以下のURLにアクセス：
   ```
   http://localhost:8000
   ```

### 方法2: Node.jsを使用

1. ターミナルを開く
2. 以下のコマンドを実行：

```bash
cd "月次定例/2026年/1月/analysis_aio/Tier1_AIO/AIO_PDCA爆速システム"
npx http-server -p 8000
```

3. ブラウザで以下のURLにアクセス：
   ```
   http://localhost:8000
   ```

### 方法3: VS CodeのLive Server拡張機能を使用

1. VS Codeで `index.html` を開く
2. 右クリック → 「Open with Live Server」
3. 自動的にブラウザで開きます

## トラブルシューティング

### 「データが読み込めない」というエラーが出る場合

1. ブラウザの開発者ツール（F12）を開く
2. Consoleタブでエラーメッセージを確認
3. 以下のファイルが存在するか確認：
   - `data/progress.json`
   - `data/baseline.json`

### ポート8000が使用中の場合

別のポート番号を使用してください：

```bash
python3 -m http.server 8080
```

その後、`http://localhost:8080` にアクセス

### まだ開けない場合

1. ブラウザのキャッシュをクリア
2. 別のブラウザで試す（Chrome推奨）
3. ブラウザのコンソールでエラーメッセージを確認

## ファイルパス

システムのファイルは以下の場所にあります：

```
月次定例/2026年/1月/analysis_aio/Tier1_AIO/AIO_PDCA爆速システム/
├── index.html          ← これをブラウザで開く（ローカルサーバー経由）
├── css/
├── js/
├── data/
└── templates/
```

---

**重要**: `file://` プロトコル（ファイルを直接開く）では動作しません。必ず `http://localhost:8000` のようにローカルサーバー経由でアクセスしてください。
