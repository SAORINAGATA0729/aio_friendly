# AIO_PDCA爆速システム

AIO対策のPDCAサイクル全体を自動化するWebアプリケーション。

## 機能概要

### 1. Plan（仮説立案）
- 現状把握ダッシュボード
- 仮説立案フォーム
- 進捗状況の可視化

### 2. Do（施策実行）
- 記事リライト機能
- チェックリストベースのAIサポート
- Markdownエディタ
- リアルタイムプレビュー

### 3. Check（検証）
- モニタリングデータ入力
- 前後比較グラフ
- 効果測定結果の表示

### 4. Action（振り返り）
- 効果測定結果の可視化
- 改善提案の記録
- レポート生成・PDF出力

## デプロイ方法

### Vercel（推奨）
1. GitHubにプッシュ
2. Vercelでリポジトリをインポート
3. 自動デプロイ完了

詳細は `GitHub_Vercelデプロイ手順.md` を参照してください。

## 使い方

### 基本的な使い方

#### Planタブ
- 現状のKPIを確認
- 仮説を立案して保存

#### Doタブ
- 記事一覧から改修したい記事をクリック
- URL入力画面で「ブラウザで開く」→記事をコピー
- 「空のエディタを開く」でエディタに貼り付け
- チェックリスト項目をクリックしてAI指示を確認
- Markdownエディタで記事を編集
- 保存ボタンで保存

#### Checkタブ
- モニタリングデータを入力（Google Search ConsoleやAhrefsから手動で入力）
- 前後比較グラフを確認

#### Actionタブ
- 効果測定結果を確認
- 改善提案を記録
- レポートを生成・PDF出力

## 技術スタック

- HTML/CSS/JavaScript（Vanilla）
- Chart.js（グラフ表示）
- html2pdf.js（PDF生成）
- Local Storage（データ保存）
